Imprimer et/ou Sauvegarder un bon de visite:

   1. Créez un bon de visite.
    2. Cliquez sur le bouton Imprimer, une nouvelle fenêtre s’ouvre contenant la vue de votre bon de visite.

  1. Impression:

    3. Cliquez sur le bouton Imprimer de la barre des tâches (le deuxième bouton), une fenêtre nommée impression s’ouvre. 

  
       


    4. Choisissez le nom de votre imprimante puis cliquez sur Propriétés.

                 

    5. Choisissez l’orientation de la page puis cliquez sur le bouton Avancé.

                  

    6. Pour modifier un paramètre cliquez dessus, une liste de choix s’affiche, maintenant vous n’avez qu’à choisir.
   7. Cliquez sur OK de toutes les fenêtres.

C’est fini.

   2. Sauvegarde:

    3. Cliquez sur le bouton Enregistrer de la barre des tâches (le premier bouton), une fenêtre nommée Enregistrer s’ouvre.


            

    4. Choisissez le dossier, le format et cliquez sur Enregistrer.

C’est fini.
