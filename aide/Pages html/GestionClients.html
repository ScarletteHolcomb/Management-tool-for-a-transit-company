1. Pour ajouter un client,  remplissez le formulaire à droite par les informations demandées puis cliquez sur le bouton ‘Ajouter’.

  2. Pour supprimer un client, vous pouvez remplissez ses informations dans le formulaire comme vous pouvez effectuer une recherche par nom ou code de client en utilisant la barre de recherche en haut puis sélectionnez le et en fin cliquez sur le bouton ‘supprimer’.

  3. Pour modifier un client, sélectionnez le dans le tableau (vous remarquez que ses informations sont copiées dans le formulaire), faites les changements nécessaires et cliquez sur le bouton ‘Modifier’.
